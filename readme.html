Installation

  1. Unzip the wordpress.jar file to a temporary WORDPRESS_PLUGIN folder.
  2. Copy all the wordpress*.jar files in the WORDPRESS_PLUGIN/classes folder to the OPENFIRE_HOME/lib folder.
  3. Copy the wordpress.jar file to the OPENFIRE_HOME/plugins folder.
  4. Restart Openfire.
  5. Configure the admin properties page.

Configuration

Under Server settings -> WordPress tab you can configure wordpress.
